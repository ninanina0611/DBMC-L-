# 简易MySQL系统
轻量级关系型数据库管理系统，基于C++开发，实现MySQL核心基础功能，满足简单的数据存储、查询、修改、删除等操作需求，兼顾易用性与轻量性，适用于小型项目数据管理与数据库原理学习场景。

## 项目简介
本项目为**简易MySQL系统**开发工程，采用分层架构设计与敏捷开发模型，由4人开发小组协作完成，核心实现轻量级关系型数据库的基础核心功能与拓展优化功能，底层基于本地二进制文件实现数据持久化，手工实现SQL语句的词法与语法解析，提供简洁的命令行交互界面。

项目整体架构分为四层，从下到上依次为**数据存储层**、**核心逻辑层**、**SQL解析层**、**交互界面层**，各层低耦合、高内聚，通过接口完成层间交互，便于模块开发、测试与维护。

## 技术栈
- **开发语言**：C++（兼具面向对象特性与执行效率，适配数据库底层开发）
- **存储方案**：本地二进制文件存储（读写快、体积小、实现轻量）
- **解析方案**：手工词法分析 + 递归下降法语法分析（灵活定制校验规则，适配基础SQL解析）
- **版本管理**：Git + GitHub（代码追溯、PR审核、版本回滚）
- **项目协同**：语雀（任务跟踪、甘特图管理、每日进度同步）
- **开发模型**：敏捷开发（1周迭代周期，增量交付、快速适配）

## 系统架构
采用分层架构设计，各层核心职责与模块如下：
### 1. 数据存储层（最底层）
系统底层数据支撑，负责数据本地持久化与加载，屏蔽存储细节，提供统一读写接口。
- 核心模块：文件操作模块、数据序列化/反序列化模块、索引管理模块、数据备份与恢复模块
### 2. 核心逻辑层（核心层）
系统业务核心，处理上层执行指令，完成数据库/数据表/数据的增删改查及约束校验。
- 核心模块：数据库管理模块、数据表管理模块、数据操作模块、约束校验模块、批量操作模块
### 3. SQL解析层（中间层）
处理用户SQL输入，完成词法/语法分析，转换为执行指令传递至核心层，提供错误提示。
- 核心模块：词法分析模块、语法分析模块、错误处理与提示模块、排序/去重解析模块
### 4. 交互界面层（最上层）
用户操作入口，实现系统与用户的交互，接收输入、反馈执行结果/错误提示。
- 核心模块：命令行输入模块、结果输出与格式化模块、界面提示与帮助模块
### 5. 跨层/扩展功能
基于各层能力实现的拓展特性，提升系统实用性与可追溯性。
- 核心模块：操作日志记录模块、索引创建/维护接口模块

**层间交互流程**：
用户命令行输入SQL → 交互界面层传递至SQL解析层 → 解析层词法/语法分析生成执行指令 → 核心逻辑层调用数据存储层完成数据操作 → 存储层返回结果至核心层 → 核心层反馈结果至交互界面层 → 界面层格式化展示给用户

## 核心功能规划
本系统功能分为**基础核心功能**与**拓展优化功能**，开发完成后将实现以下能力：
### 基础核心功能
1. 数据库基础操作：建库/删库/选库、建表/删表/修改表结构（支持字段约束）
2. 数据CRUD：实现INSERT/DELETE/UPDATE/SELECT，支持单表条件查询
3. 数据持久化：本地文件存储，程序重启后可加载数据
4. SQL语法校验：词法/语法错误、对象不存在等精准提示
5. 命令行交互：支持多行输入、历史命令、结果格式化展示
### 拓展优化功能
1. 简单索引：主键索引创建与维护，提升查询效率
2. 查询优化：支持ORDER BY排序、DISTINCT去重
3. 批量操作：批量插入/修改，提升大批量数据处理效率
4. 操作日志：记录核心操作与时间，支持问题追溯
5. 数据备份：本地备份文件导出/导入，防止数据丢失
